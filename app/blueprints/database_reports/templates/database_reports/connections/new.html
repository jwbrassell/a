    Connection Help
    Oracle Connection Format

    Host: The hostname or IP address of your Oracle server

    Port: Usually 1521

    Database: The service name or SID

    Example:

    Host: oracle.example.com
Port: 1521
Database: ORCL
    MySQL Connection Format

    Host: The hostname or IP address of your MySQL server

    Port: Usually 3306

    Database: The database name

    Example:

    Host: mysql.example.com
Port: 3306
Database: myapp_db
    SQLite Connection Format

    Database: Full path to the SQLite database file

    Example:

    Database: /path/to/database.db

    Note: Host and port are not used for SQLite connections.

    Security Note

    Credentials are stored securely in vault and are never exposed in the application. Make sure you have the necessary permissions to access the database.
